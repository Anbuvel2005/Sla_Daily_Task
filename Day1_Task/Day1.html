HTML Introduction

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It helps browsers display content such as headings, paragraphs, images, links, tables, and forms.

HTML was created by Tim Berners-Lee in 1991 and HTML5 is the latest version used for modern web development.

HTML Structure

  • <!DOCTYPE html> - Tells the browser that the document is an HTML5 document.
  • <html> - The root element that contains all the content of the webpage.
  • <head> - Contains metadata such as title, styles, and character settings.
  • <body> - Contains all visible content displayed on the webpage.
